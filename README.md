
Come giocare
Esegui il file main.py con Python:
python main.py

Poi scegli un’azione tra quelle disponibili:
 
1  Lavorare → guadagni 110 euro.
2  Investire → costa 50 euro, guadagno casuale.
3  Fare la spesa → costa 40 euro, aumenta la fame.
4  Comprare grano → costa 15 euro per unità.
5  Vendere grano → guadagni 15 euro per unità.
6  Uscire → termina il gioco.

Obiettivo
  Sopravvivere il più a lungo possibile senza far scendere la fame a 0.
  Se la fame arriva a 0, il gioco finisce.
